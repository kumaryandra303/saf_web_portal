# Settibalija Action Force (SAF) Website

Modern, professional website for the Settibalija Action Force community welfare organization.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Modern React + Vite setup
- Tailwind CSS styling
- Responsive design
- Multiple pages (Home, About, Programs, Updates, Gallery, Contact)
- Social media integration
- Professional UI components

## Color Palette

- SAF Red: #dc2626 (primary brand color from logo)
- SAF Dark: #1a1d20 (text and backgrounds)
- SAF Gold: #eab308 (accents)
